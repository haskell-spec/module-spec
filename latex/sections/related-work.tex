\begin{itemize}
    \item \citet{Faxen2002staticsemantics} specifies the complete static semantics of Haskell 98, and as a part of the static semantics also formalizes the module system.
     That paper diverges from the official report, however.
    \item The paper \citet{Diatchki2002haskellmodules} gives an executable specification of the semantics of the Haskell 98 module system in Haskel.
    \item The Haskell 2010 language report \citet{Haskell2010} is the current official specification of Haskell.
\end{itemize}
